# DevFlow

DevFlow is a Next.js project management app with:

- authentication
- project creation and update
- subtask management with due dates and priority
- dashboard analytics
- AI-generated subtask suggestions with Gemini

## Run locally

```bash
npm install
npm run dev
```

App runs at `http://localhost:3000`.

## Main folders

- `app` - pages and API routes
- `modules` - feature logic for auth, projects, and AI
- `components` - reusable UI pieces
- `prisma` - database schema
